Ajuda
Dicas para treinar o programa:
  1. Comece com faturas claras: Escolha faturas com formato simples e legível.
  2. Selecione com precisão: Tente selecionar apenas o texto relevante para cada campo.
  3. Treine com múltiplos exemplos: Quanto mais faturas treinar, melhor o programa fica.
  4. Verifique os resultados: Depois de treinar, teste a extração para ver se está correto.
  5. Corrija erros: Se o programa extrair dados errados, adicione mais exemplos de treino.

Nota: Os padrões aprendidos são guardados no ficheiro training/patterns.json. Pode exportar e importar este ficheiro para partilhar com outros utilizadores.
