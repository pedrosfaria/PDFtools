Sobre o PDF Invoice Extractor

O que é?

O PDF Invoice Extractor é uma aplicação que permite extrair automaticamente dados de faturas de eletricidade em formato PDF.

O sistema identifica o fornecedor (EDP, Galp, Ibersol, Coopérnico, etc.) e extrai informações como consumo, valores, datas, dados do cliente e muito mais.

Funcionalidades
  • Extração automática: Processa ficheiros PDF e extrai dados estruturados
  • Múltiplos fornecedores: Suporta EDP, Galp, Ibersol, Coopérnico e outros
  • Múltiplos formatos: Exporta para CSV, Excel ou JSON
  • Processamento em batch: Processa múltiplos ficheiros de uma vez
  • OCR opcional: Suporta faturas digitalizadas (scans) com Tesseract OCR
  • Interface web: Interface intuitiva para upload e visualização
Tecnologias Utilizadas
Backend:
  • Python 3.8+ - Linguagem principal
  • Flask - Framework web
  • pdfplumber - Extração de texto de PDFs
  • pandas - Manipulação de dados
  • openpyxl - Criação de ficheiros Excel
  • pytesseract - OCR para ficheiros com imagens
Frontend:
  • Bootstrap 5 - Framework CSS
  • Font Awesome - Ícones
  • Jinja2 - Templates
Dados Extraídos

O sistema extrai os seguintes campos das faturas:

Informações da Fatura
  • Fornecedor
  • Número da fatura
  • Data de emissão
  • Data de vencimento
Dados de Consumo
  • Período de consumo
  • Consumo em kWh
  • Potência contratada
  • Preço por kWh
Valores Monetários
  • Custo da energia
  • Custo de acesso à rede
  • Taxa de IVA
  • Valor do IVA
  • Total a pagar
Cliente
  • Nome
  • NIF
  • Número do cliente
  • Morada
  • Código postal
  • Localidade
Contador
  • Número do contador
  • Leitura atual
  • Leitura anterior
  • Data da leitura
Pagamento
  • Método de pagamento
  • Referência de pagamento
